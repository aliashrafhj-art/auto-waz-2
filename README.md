# Auto Waz 2.0 🚀

Drive → YouTube Auto Scheduler

## Railway Environment Variables

```
YT_CLIENT_ID=your_google_client_id
YT_CLIENT_SECRET=your_google_client_secret
BASE_URL=https://your-app.railway.app
```

## Features
- প্রতিদিন আলাদা Drive Folder
- Multiple time slots
- Per-day title/caption template
- Auto token refresh (প্রতি ৩০ মিনিট)
- Config + Token Drive backup
- Video queue (FIFO cycle)

## Setup
1. Railway তে deploy করো
2. Environment variables দাও
3. YouTube + Drive connect করো
4. প্রতিদিনের Folder ID দাও
5. Time slots add করো
6. Toggle চালু করো ✅
